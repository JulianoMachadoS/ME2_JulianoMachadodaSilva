==== OBJETIVO DO TRABALHO ====

- aprendizado com métodos e modificações de Strings e cálculos matemáticos

====PLANEJAMENTO====
- Implementação de um menu interativo para a navegação entre as funcionalidades do programa.
- Desenvolvimento da funcionalidade de adição e posterior alteração de nomes.
- Desenvolvimento da funcionalidade de adição e posterior alteração de datas.
- Inclusão das opções "Limpar Tela" e "Sair do Programa" no menu principal.

==== O QUE FOI COLOCADO ====
- Descobrir qual o seu signo do zodíaco informando dia e mês do seu nascimento.
- Dizer qual é a sua idade (anos completos, meses e dias) considerando a data da consulta.
- Descobrir o dia da semana que a pessoa nasceu usando o algoritmo de Zeller.
- Contar a quantidade de vogais do nome completo a ser inserido pelo usuário
    (resolver a questão do espaço entre os nomes).
    Escrever o nome lido e a quantidade de vogais e consoantes deste nome.
- Montar um anagrama de um nome lido (considere apenas um nome simples e não composto).
- Dentro da edição do nome foram feitos os métodos para saber quantas vogais, consoantes e
    letras tem o nome e se o nome é um palíndromo criando um anagrama com o mesmo.
- SISTEMA DE HISTÓRICO:
       - Armazenamento de múltiplas pessoas com todas suas informações
       - Visualização completa do histórico com dados formatados
       - Remoção de pessoas do histórico por ID
       - Atualização automatica de registros quando os dados são alterados
       - Sistema de IDs sequenciais para facilitar gerenciamento
       - Salvamento automático ao adicionar data quando nome já está preenchido
- Sair do programa.
- Posibilidade de limpar a tela.

==== DESAFIOS TECNICOS SUPERADOS ====

Manipulação de Strings sem Vetor ou Array's:
   - Todos os contadores e dados armazenados como Strings
   - Operações matemáticas através de conversões String para int e int para String
